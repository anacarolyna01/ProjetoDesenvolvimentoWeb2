O que é o IFBus?

O IFBus é uma aplição web cujo o principal objetivo é permitir o melhor gerenciamento das frotas de onibus.


O que podemos fazer no IFBus?

Essa aplicação permite o cadastro dos motoristas e dos onibus, além de permitir a consulta de dados sobre os mesmos.
